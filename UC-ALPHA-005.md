# UC-ALPHA-005 — Wave 2 Crypto Pump (CEX Listed)

**Timeframe:** Hours to 3 days

## Thesis

This mirrors the Wave 2 meme coin pattern but applies specifically to assets listed on centralized exchanges, where deeper order book liquidity and regulatory listing standards meaningfully reduce — though do not eliminate — the risk profile compared to decentralized exchange launches. The thesis is identical: a confirmed first wave of momentum followed by a healthy consolidation sets up favorable conditions for a second leg, with the centralized exchange listing providing better execution and lower rug-pull risk than an unverified token contract.

## Entry Signal

Same core pattern as the DEX version — confirmed Wave 1, healthy consolidation holding above the prior base — but with the additional confirmation that the asset trades on a reputable, regulated exchange with genuine order book depth. Volume and liquidity conditions are verified directly from exchange data rather than inferred from on-chain activity alone.

## Exit Rules

Same target and stop logic as the DEX version, but tighter given the better liquidity profile — exits can be executed with less slippage, which allows for more precise risk management. The time stop remains the same: no meaningful movement within 24 hours suggests the setup has failed.
